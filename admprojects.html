Interactive Quiz-Based Activity

October 2023

Using free game development platform Roblox Studio, I created an interactive quiz-based activity/'game' for my Graduate Writing Assessment (GWAR) course, ADM360GW: 'Fashion, Clothing, and Society'. This was intended to accompany a 4-person group presentation on an assigned chapter in the course's textbook. Users that logged on would be able to answer the 4-question quiz related to the chapter, following the game objectives of correctly answering each question to obtain the items needed to 'win' the game.
